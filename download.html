    Obtain phpDaemon

    Sources


    GitHub: github.com/kakserpom/phpdaemon

    At this moment, the best practice to obtain phpDaemon is cloning the repository:
    $ git clone https://github.com/kakserpom/phpdaemon.git

    Do not forget to install:

    1. PHP >= 5.4 non-zts
    2. libevent 2.last and pecl-event 1.6.0
    3. pecl-eio to increase filesystem I/O performance
